# UserAuthSystem

Full-stack OTP-based user auth:
- Node.js, Express, MongoDB, JWT, Bcrypt, Nodemailer
- React (Vite), React Bootstrap, Toastify

## Run locally
1. Start MongoDB
2. Backend:
cd server
cp .env.example .env # if you add one
npm install
npm run dev
3. Frontend:
cd ../client
npm install
npm run dev
